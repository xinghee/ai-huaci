隐私权政策

AI 划词助手是一款 Chrome 扩展，用于在用户选中网页文字后，提供翻译、总结、解释、复制和朗读等辅助功能。我们重视用户隐私，并尽量让数据处理保持透明、可控和最小化。

1. 我们处理哪些数据

  • 用户主动选中的网页文本：仅在用户点击扩展工具栏中的翻译、总结、解释等按钮后，才会用于执行相应功能。
  • AI 服务配置：包括用户自行填写的 API Base URL、API Key、模型名称等，用于连接用户选择的 OpenAI 兼容 AI 服务。
  • 扩展设置：包括按钮配置、提示词、外观设置、朗读引擎设置等，用于保存用户偏好。
  • 浏览器权限状态：当用户授权扩展在特定网站运行时，扩展会使用该授权在相关网页上显示划词工具栏。

2. 我们如何使用数据

选中文本只会在用户明确点击某个功能按钮后被处理。若用户使用 AI 翻译、总结、解释等功能，选中文本会被发送到用户在设置页中配置的第三方 AI API 服务。AI 划词助手本身不运营用于接收这些文本的服务器。

API Key、Base URL、提示词和扩展设置保存在 Chrome 扩展存储中，用于实现扩展的核心功能。朗读功能可能使用浏览器内置的语音合成能力处理文本。

3. 数据共享与出售

我们不会出售用户数据，不会将用户数据用于广告追踪，也不会将用户数据共享给数据经纪商。用户主动配置的第三方 AI 服务可能会根据其自身隐私政策处理请求内容，请用户自行确认所使用服务的条款。

4. 数据存储

扩展设置和 API 配置保存在 Chrome 扩展存储中。用户可以在扩展设置页修改相关配置，也可以通过卸载扩展或清除浏览器扩展数据来删除这些信息。

5. 权限说明

AI 划词助手使用 Chrome 扩展存储权限保存设置。扩展可能请求网站访问权限，以便在用户选中文字时显示工具栏并执行用户点击的操作。扩展不会在用户未选中文字并点击功能按钮的情况下主动发送网页内容。

6. 用户选择与控制

  • 用户可以不配置 AI API，仅使用复制或浏览器朗读等本地功能。
  • 用户可以随时修改或删除 API Key、Base URL、提示词和按钮设置。
  • 用户可以在 Chrome 浏览器中撤销扩展的网站访问权限或卸载扩展。

7. 联系方式

如对本隐私权政策有疑问，可以通过 GitHub 仓库提交 Issue：https://github.com/xinghee/ai-huaci

Privacy Policy

AI Selection Assistant is a Chrome extension that helps users translate, summarize, explain, copy, and read aloud selected webpage text. We aim to keep data handling transparent, user-controlled, and limited to what is necessary for the extension's core features.

1. Data We Process

  • Text selected by the user on webpages: this text is processed only after the user clicks a feature button such as translate, summarize, explain, or read aloud.
  • AI service settings: this may include the API Base URL, API Key, model name, and related settings entered by the user to connect to an OpenAI-compatible AI service.
  • Extension settings: this may include button configuration, custom prompts, appearance preferences, and text-to-speech settings.
  • Site permission status: when the user grants site access, the extension uses that permission to display the selection toolbar on relevant webpages.

2. How We Use Data

Selected text is processed only when the user explicitly clicks a feature button. If the user uses AI-powered features such as translation, summarization, or explanation, the selected text is sent to the third-party AI API service configured by the user in the extension settings. AI Selection Assistant does not operate its own server to receive or store selected text.

API keys, Base URLs, prompts, and extension preferences are stored in Chrome extension storage so the extension can provide its core features. The read-aloud feature may use the browser's built-in speech synthesis capability to process text locally in the browser.

3. Sharing and Sale of Data

We do not sell user data, use user data for advertising tracking, or share user data with data brokers. Third-party AI services configured by the user may process request content according to their own privacy policies. Users should review the terms and privacy policies of the AI services they choose to use.

4. Data Storage

Extension settings and API configuration are stored in Chrome extension storage. Users can modify these settings on the extension options page, uninstall the extension, or clear browser extension data to remove stored information.

5. Permissions

AI Selection Assistant uses Chrome storage permission to save settings. The extension may request website access permission so it can display the selection toolbar and perform user-triggered actions on webpages. The extension does not send webpage content unless the user selects text and clicks a feature button.

6. User Choice and Control

  • Users may choose not to configure an AI API and only use local features such as copy or browser read-aloud.
  • Users can modify or delete their API Key, Base URL, prompts, and button settings at any time.
  • Users can revoke site access permissions in Chrome or uninstall the extension.

7. Contact

For questions about this Privacy Policy, please open an Issue in the GitHub repository: https://github.com/xinghee/ai-huaci

The extension's single purpose is to perform user-selected actions on text that the user explicitly selects. We process only the data needed to provide those features.
